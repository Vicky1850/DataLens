# Multi-Excel Search

Search across multiple Excel files instantly from your browser.

## Quick Start

### Option 1 — Python (recommended, no install needed)
```
python server.py
```
Then open: http://localhost:8080

### Option 2 — Double-click (Windows)
Double-click `start.bat`

### Option 3 — Open directly
Just open `index.html` in your browser.
> Note: Some browsers block file:// access for drag-and-drop. Use the server if that happens.

## How to use
1. Drag & drop up to 10 Excel files (.xlsx / .xls) onto the upload area
2. Type any value in the search box
3. Filter by column or file as needed
4. Click "Export CSV" to save your results
